# KidLearn

Interactive math and reading app for kids!

## Features
- **Math:** Counting, Addition, Subtraction with visual aids
- **Reading:** Letters A-Z, Sight Words, Phonics word building
- **Exercises:** Multiple choice, tap-to-count, fill-in-blank, drag-and-drop
- **Rewards:** Stars, streaks, badges

## Development

```bash
npm install
npm run dev
```
